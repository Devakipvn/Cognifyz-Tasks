Interactive JavaScript Features

In This Website i made the interactive changes based on the JavaScript Features to enhance the website.

Change Button Color

A button that changes its background color when clicked.

This is achieved using JavaScript event listeners and inline style manipulation.

Greeting Based on Time

A button displays a greeting message like "Good Morning", "Good Afternoon" or "Good Evening" based on the current time of the day.

This enhances user interactivity by responding dynamically to the time.

Simple Calculator

Allows users to input two numbers, and upon clicking the "Add Numbers" button, the sum is displayed.

Validates the input to ensure it only accepts numbers
